# 日次記事生成プロンプト

以下の手順で Zenn の技術記事を1本作成してください。

## テーマ選定

以下のカテゴリからローテーションで1つ選び、具体的なテーマを決めてください：

1. **Claude Code 実践Tips** - 便利なコマンド、ワークフロー、設定のコツ
2. **AI駆動開発** - Claude Code を使った開発手法、プロンプトエンジニアリング
3. **開発効率化** - Git, Shell, エディタ等の生産性向上テクニック
4. **Web技術トレンド** - TypeScript, React, Next.js 等の最新情報
5. **インフラ・DevOps** - Docker, CI/CD, クラウドサービスの活用法

## 既存記事の確認

`articles/` ディレクトリ内の既存記事を確認し、テーマが重複しないようにしてください。

## 記事作成の手順

1. テーマに基づいた slug を決める（例: `claude-code-git-workflow-tips`）
2. 今日の日付（YYYY-MM-DD）を slug の末尾に付ける
3. `articles/{slug}.md` に記事を作成する
4. CLAUDE.md の記事作成ルールに従う

## 記事の構成テンプレート

```
## はじめに
（課題や背景を簡潔に）

## 本題
（具体的な手順・コード・解説）

## まとめ
（要点の整理・次のステップ）
```

## 完了条件

- [ ] 記事ファイルが `articles/` に作成されている
- [ ] Frontmatter が正しい形式である
- [ ] `published: false` になっている（手動で確認後に公開する）
- [ ] 変更を git commit している
