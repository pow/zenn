# GraphQL カーソルベースページネーション（Go 実装）

記事「Go で GraphQL のカーソルベースページネーションを実装する」のサンプルコードです。

## セットアップ

Go 1.22 以上が必要です。

## ファイル構成

- `cursor.go` — カーソルの Base64 エンコード/デコード
- `paginate.go` — 汎用ページネーション関数（ジェネリクス）
- `cursor_test.go` — カーソル関数のテスト
- `paginate_test.go` — ページネーション関数のテスト

## テスト実行

```bash
go test -v ./...
```
